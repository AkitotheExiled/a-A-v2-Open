My HTML Cheatsheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`.

Paragraphs

Paragraphs of text go in `p` tags.

Lists

Types of lists:

  • Unordered Lists `li`
  • Ordered Lists `li`

Unordered Lists

Unordered lists are marked with bullet points and use the tag `ul`

Ordered Lists

Ordered lists use numbers and go in `ol` tags. Each list item is numbered and goes in order

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan
  3. Carry out the plan
  4. Go back and improve your solution

Anchors(links)

Anchor tags create links to other pages. The link goes within the href attribute. Search the web for more information

Setting `target="_blank"` will open up the page in a new window.

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within the tags

Bold

You can also `strong` element to emphasize text. This shows up as bold text in most browsers.

Organizing Content with `div` and `span`

`div`

The `div` element is a block element. So each `div` will show up on new line.

`span`

The `span` element is an inline element, so there will be no line break between each `span`
